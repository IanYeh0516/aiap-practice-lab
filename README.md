# AIAP 試煉場

AI 應用規劃師本機優先的練習網站，提供解析練習、歷屆考題隨機組卷與永久錯題本。

## 線上版本

網站由 GitHub Pages 自動部署。建立完成後，可從 GitHub 專案頁面的「部署」或「設定 → Pages」開啟正式網址。

## 功能

- 初級、中級及混合組卷
- 解析練習與歷屆考題模式
- 5、10、20、30 題隨機測驗
- 計時、交卷、成績及逐題檢討
- 使用瀏覽器本機儲存空間保存錯題；連續答對兩次後移除
- 響應式桌面與手機版面

## 公開資料來源

題目、學習指引及官方答案整理自以下公開資料：

- [經濟部產業發展署 iPAS－AI 應用規劃師學習資源](https://ipd.nat.gov.tw/ipas/certification/AIAP/learning-resources)

資料類型包含初級與中級學習指引、歷屆公告試題及主辦單位提供的外部學習資源。

## 使用聲明

本站為非官方練習工具，內容可能受到原始文件排版或人工整理影響，題目、答案、考試範圍及制度資訊應以主辦單位最新公告為準。

學習指引原有解析與 Codex 補充解析會分開標示；AI 補充內容不是官方解析。

歷屆題的補充解析會先由 Codex 獨立解題，再以同級、同科目的官方學習指引 PDF 進行第二輪驗證。只有獲得教材直接支持的內容會標記為來源已驗證；教材未涵蓋或原始題目排版異常者會保留待複核狀態。

## 本機執行

```bash
python3 -m http.server 4173 --directory site
```

開啟 <http://127.0.0.1:4173>。
