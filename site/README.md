# AIAP 試煉場

本機靜態題庫網站，整合五份學習指引解析題與十二份官方歷屆考題。

## 公開資料來源

- [經濟部產業發展署 iPAS－AI 應用規劃師學習資源](https://ipd.nat.gov.tw/ipas/certification/AIAP/learning-resources)

本站為非官方練習工具，題目與答案應以主辦單位最新公告為準。

## 使用方式

macOS 可直接雙擊專案根目錄的 `開啟題庫網站.command`。

也可以直接開啟 `site/index.html`。若瀏覽器限制本機腳本，可在專案根目錄執行：

```bash
python3 -m http.server 4173 --directory site
```

然後開啟 `http://127.0.0.1:4173`。

## 題庫更新

當題庫來源更新後，在專案根目錄執行：

```bash
python3 scripts/extract_questions.py
```

程式會重新產生 `site/data.js`。十二份歷屆試卷已完整整理為 600 題；官方 PDF 沒有逐題解析，因此由 Codex 逐批補充解析。產生器會自動把已完成解析併入網站，並保留官方答案與 AI 補充解析的來源區隔。

## 解析與教材驗證流程

歷屆題解析採兩階段處理：

1. Codex 先獨立判斷核心原理、正解理由、四個選項與專業術語。
2. 再檢索同級、同科目的官方學習指引 PDF，以檔名及 PDF 頁碼驗證答案與解析。

直接獲得教材支持的題目標記為「來源已驗證」；教材未涵蓋、題幹缺漏、跨題黏接或內容衝突者保留為待複核，不會以 AI 推論冒充官方解析。

目前 600 題歷屆題皆已完成 Codex 逐題解析與 PDF 二次查證；其中依賴圖表、程式碼或前題資料的 26 題也已人工檢視官方公告試題原頁。教材與公告答案無法一致的題目會保留衝突說明，不會自動改寫公告答案。
